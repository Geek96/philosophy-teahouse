# 尼采生平时间线

| 时间 | 事件 |
|---|---|
| 1844.10.15 | 生于普鲁士萨克森的勒肯(Röcken),父亲卡尔·路德维希是路德宗牧师 |
| 1849 | 父亲病逝(脑软化症),次年幼弟约瑟夫夭折;举家迁往瑙姆堡,由母亲、祖母、姑姑抚养长大,家中女性环绕 |
| 1858–1864 | 就读普夫达中学(Schulpforta),接受严格古典语文学训练 |
| 1864–1869 | 波恩大学、莱比锡大学攻读古典语文学与神学;1865年前后读叔本华《作为意志和表象的世界》,深受震动 |
| 1868 | 在莱比锡结识瓦格纳 |
| 1869 | 年仅24岁被聘为巴塞尔大学古典语文学教授(未获博士学位即破格任教);此后常客居瓦格纳在特里布森的别墅 |
| 1870 | 短暂服役普法战争担任救护兵,染病(疑似白喉/痢疾),健康自此持续恶化 |
| 1872 | 出版《悲剧的诞生》,融合日神/酒神二元论,为瓦格纳歌剧摇旗呐喊,遭学界冷遇 |
| 1876 | 拜罗伊特音乐节后与瓦格纳关系急剧降温,不满其转向基督教叙事与德意志民族主义 |
| 1878 | 出版《人性的,太人性的》,标志与瓦格纳公开决裂 |
| 1879 | 因严重偏头痛、眼疾、肠胃病等多重病痛辞去巴塞尔教职,此后靠微薄退休金四处漂泊(瑞士锡尔斯玛利亚、意大利热那亚/都灵等地) |
| 1881–1885 | 漂泊高产期:《朝霞》(1881)、《快乐的科学》(1882)、《查拉图斯特拉如是说》(1883–1885) |
| 1882 | 与莎乐美(Lou Salomé)相恋受挫,是其一生情感创伤之一 |
| 1886–1888 | 《善恶的彼岸》(1886)、《道德的谱系》(1887)、《偶像的黄昏》《反基督者》《瞧,这个人》等(1888) |
| 1889.1.3 | 在都灵街头目睹马匹遭鞭打而抱其颈痛哭,随即精神崩溃,写出多封"疯狂书信" |
| 1889–1900 | 精神失常长达11年,先后由母亲、后由妹妹伊丽莎白·福斯特-尼采照管,基本丧失语言与自理能力 |
| 1900.8.25 | 因中风并发肺炎在魏玛去世 |
| 1894起 | 伊丽莎白设立"尼采档案馆",掌管全部遗稿;后被指删改、拼凑《权力意志》等文本,并与丈夫伯恩哈德·福斯特的反犹立场牵连,使尼采长期被误读、被纳粹意识形态挪用 |
| 1930s–1960s | 卡尔·洛维特、瓦尔特·考夫曼等学者陆续澄清尼采思想与纳粹主义的本质区别;1960年代蒙提那里、科利主持的德文考订版(KGW/KSA)据原始手稿重新校勘,清除伊丽莎白篡改痕迹,尼采才在学界获得正名 |

**近年研究/接受史动向**:巴黎-柏林合作的"尼采数字资源"项目持续推进《尼采全集校勘版》数字版(eKGWB, doc.nietzschesource.org)与"尼采遗稿数字影印版"(DFGA),首次完整收录约6600处此前分散于各版本的语文学校勘意见,并按手稿原始页面拓扑结构逐页转录1885–1889年晚期遗稿,便于研究者绕开伊丽莎白时代编纂顺序、直接核对手稿本貌;柏林-勃兰登堡科学与人文学院"晚期尼采"项目亦在持续推进相关校勘与研究。
