# 加缪生平时间线

## 关键节点表

| 年份 | 事件 |
|---|---|
| 1913.11.7 | 生于法属阿尔及利亚蒙多维(今德雷昂),移民家庭;父亲鲁西安为农业工人 |
| 1914 | 一战爆发,父亲随祖阿夫团参战,当年即阵亡;加缪随母迁往阿尔及尔贝尔库贫民区,与失聪、近乎文盲的母亲及外祖母、舅舅同住 |
| 1923 | 小学教师路易·热尔曼发现其天赋,助其获奖学金进入中学 |
| 1930 | 17岁确诊肺结核,健康问题伴随终生,一度中断学业与足球运动(曾任阿尔及尔大学队守门员) |
| 1932–1936 | 就读阿尔及尔大学,主修哲学,师从让·格勒尼耶;1935年短暂加入法国共产党,后转入阿尔及利亚共产党,1936年退出 |
| 1935–1939 | 组织"劳工剧院"等剧团;担任《阿尔及尔共和报》记者,报道卡比利亚饥荒等社会议题 |
| 1940 | 移居巴黎,任《巴黎晚报》编辑;与西蒙娜·耶后离婚,同年与弗朗辛·福尔结婚 |
| 1942 | 出版《局外人》与哲学随笔《西西弗神话》,确立"荒诞哲学"文学地位 |
| 1943–1945 | 加入抵抗组织"战斗"(Combat),任地下报纸主编,巴黎解放后公开身份,持续主笔至1947年 |
| 1947 | 出版长篇小说《鼠疫》,获评论界及读者广泛认可 |
| 1951 | 出版哲学论著《反抗者》,系统批判历史暴力与革命乌托邦 |
| 1952 | 因《反抗者》遭萨特主编的《现代》杂志(弗朗西斯·让松执笔)批评"服务于苏联辩护术";加缪致萨特公开信反击,萨特回应长文終结二人友谊,史称"加缪-萨特决裂" |
| 1956 | 出版中篇小说《堕落》 |
| 1957 | 10月获诺贝尔文学奖,43岁,为史上第二年轻的文学奖得主、首位出生于非洲的得主;颁奖词肯定其作品"清醒地照亮人类良知面临的问题" |
| 1960.1.4 | 在法国桑斯附近维勒布勒万,与出版人米歇尔·伽利玛同乘汽车时车祸身亡,年46岁;随身携带未完成手稿《第一个人》 |
| 1994 | 遗孀弗朗辛与女儿卡特琳娜整理出版自传体遗稿《第一个人》,揭示其阿尔及利亚贫困童年与母子情感,引发对其生平与思想的重新评估 |

## 近年研究与接受史动向(替代"近12个月动态")

- **移灵先贤祠的争议持续发酵**:2009–2010年萨科齐总统曾提议将加缪灵柩迁入先贤祠,遭其子让·加缪强烈反对("这将是一种荒谬"),女儿卡特琳娜态度暧昧,该提案至今未获落实,每逢逝世纪念日仍被媒体重提为"未竟之事"。
- **后殖民视角重读成为主流学术议题**:围绕《局外人》中"被杀的阿拉伯人"始终无名、失语这一叙事装置,后殖民批评(如David Carroll《Albert Camus the Algerian》)指出其虽反对法阿双方针对平民的暴力、主张多元民主的阿尔及利亚,却也因此被左翼阵营与后殖民批评者贴上"殖民同情者"标签;卡迈勒·达乌德《默尔索案调查》(2013)以阿拉伯受害者兄弟视角"反写"《局外人》,持续引发关于殖民叙事与"去命名"手法的学术讨论(2024年*Nottingham French Studies*仍有专文分析两部文本的互文关系)。
- **《第一个人》遗稿的持续影响**:该未完成自传体小说被视为理解加缪"贫困移民之子"身份与其阿尔及利亚立场矛盾性的关键文本,近年传记与后殖民研究常以此书为切入点,重新评估他在法阿冲突中"拒绝选边"立场的伦理复杂性。
