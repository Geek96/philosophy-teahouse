# 孔子生平时间线

## 关键节点表

| 时间(约) | 事件 | 可信度/来源 |
|---|---|---|
| 前551年 | 生于鲁国陬邑(今山东曲阜),父叔梁纥(陬邑大夫,武士出身),母颜徵在 | 中高。生年据《史记·孔子世家》及后世历法推算(一说前552年),《春秋》《谷梁传》记载略有出入,细节(如"野合而生")争议较大 |
| 前549年(约3岁) | 父叔梁纥去世,家道中落,随母移居曲阜阙里 | 中,《史记》及《孔子家语》记载,细节零散 |
| 前535年(约17岁)前后 | 母颜徵在去世;早年家贫,"吾少也贱,故多能鄙事"(《论语·子罕》),曾任委吏(管仓库账目)、乘田(管畜牧) | 中高,委吏、乘田见《孟子》《史记》,《论语》自述为一手史料但为追忆性质 |
| 前535-前500年左右 | 娶亓官氏,育有一子(孔鲤)一女;约30岁"三十而立",开始收徒讲学,倡"有教无类" | 中,《论语》《史记》均有记载,具体授徒起始年份古今学者推算不一 |
| 前517年(约35岁) | 鲁国内乱(三桓与鲁昭公之争),孔子一度赴齐国,闻《韶》乐"三月不知肉味" | 中高,《论语·述而》《史记》均载,属较可信的一手/近一手记载 |
| 前501年(约51岁) | 出仕鲁国,先后任中都宰、司空、大司寇,据传曾"摄行相事"(代理国相职权),夹谷会盟中助鲁定公挫齐国之谋 | 中。"摄相事"及夹谷之会见《史记·孔子世家》《谷梁传》,但记载有夸大孔子政治功绩的倾向,现代史家对具体官职权限存疑 |
| 前497年(约55岁) | 因鲁国执政季桓子受齐国女乐、怠于政事,孔子去鲁,开始周游列国 | 高,《论语·微子》《史记》均记载,为孔子生平转折的核心事件 |
| 前497-前489年 | 周游卫、曹、宋、郑、陈、蔡、楚等国,屡遭困厄:宋司马桓魋欲加害("天生德于予,桓魋其如予何");陈蔡之间绝粮,弟子多有怨言 | 中高,主要见《论语》《史记·孔子世家》《庄子》(寓言化程度高),核心事件(去陈绝粮)可信度较高,具体细节和对话多为后世追述加工 |
| 前484年(约68岁) | 在弟子冉有斡旋及季康子邀请下返回鲁国,不再出仕,转而专注教育与整理典籍(传删《诗》《书》,定礼乐,序《易》,作/修《春秋》) | 中高,返鲁时间见《左传》哀公十一年,较为可靠;但"删诗书定礼乐"等编纂功绩,现代文献学多认为是后世儒家追认、层累建构的成分较大 |
| 前483-前479年 | 晚年丧子(孔鲤先卒)、丧爱徒颜回、子路(死于卫国内乱),孔子悲痛;continue讲学 | 中,《论语》《史记》记载,时间顺序细节古今考证略有分歧 |
| 前479年 | 卒于鲁国,年73岁(一说72岁),葬于曲阜城北泗水之上(今孔林) | 高,《春秋》《左传》哀公十六年及《史记》均有明确记载,为孔子生平中最无争议的时间节点之一 |

## 近年孔子/儒学研究的最新学术动向(替代"近12个月动态")

孔子作为历史人物无当代动态,此处代之以出土文献对早期儒家研究的持续影响。1993年出土的郭店楚简(1998年正式刊布)与随后面世的上海博物馆藏战国楚竹书(上博简)、清华大学藏战国竹简、安徽大学藏战国竹简等,被学界视为"开辟出土文献新纪元"的标志性材料,其中儒家类文献数量最多。相关研究显著改变了以往仅以传世本《论语》作为研究孔子思想唯一或主要依据的格局:学者现可直接利用郭店简《缁衣》《五行》等篇、上博简中记录孔子与弟子对话及"孔门七十子"相关的佚篇,补充和校验孔子思想的早期面貌,并重新审视子思一系与孔子思想的传承关系。与此同时,《论语》成书年代与层累编纂过程(不同篇章成于不同时期、经历战国至汉代多次编定)仍是持续的研究热点,尚无定论。整体上,2020年代以来出土简牍与传世文献的对读、比勘,是孔子及早期儒家研究最活跃的方向。
