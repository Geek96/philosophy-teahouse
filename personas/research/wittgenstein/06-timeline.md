# 维特根斯坦生平时间线

来源与可信度说明:主体依据 Stanford Encyclopedia of Philosophy「Ludwig Wittgenstein」词条(学术权威,可信度高)及维基百科交叉核对基本生卒/求学年份(可信度中,仅作日期校验用);近年学术动向部分依据 Wittgenstein Archives at the University of Bergen (WAB) 官网 wab.uib.no 及 Wittgenstein Initiative 官网的项目通报(一手机构信息,可信度高)。未采用知乎、微信公众号、百度百科等来源。

## 关键节点表

| 时期 | 事件 |
|---|---|
| 1889 | 生于维也纳,富裕犹太裔工业家族(家族后皈依天主教) |
| 1906–1908 | 柏林学习机械工程 |
| 1908–1911 | 曼彻斯特大学攻读航空工程,研究螺旋桨设计;因对数学基础的兴趣转向逻辑,经弗雷格指点 |
| 1911 | 赴剑桥师从罗素,进入哲学/数理逻辑研究 |
| 1913 | 父亲去世,继承巨额财产 |
| 1914–1918 | 一战服役(奥匈军队),1918年在意大利被俘;战时及战俘营中写作笔记本(1914–1916),完成《逻辑哲学论》手稿 |
| 1918–1920 | 散尽个人财产 |
| 1920–1926 | 任奥地利乡村小学教师 |
| 1921/1922 | 《逻辑哲学论》先后以德文、英文出版 |
| 1926–1928 | 为姐姐玛格丽特设计建造维也纳住宅(建筑师阶段) |
| 1929 | 重返剑桥,认为早期哲学问题需重新彻底思考,开启后期哲学转向 |
| 1930s–1940s | 任教剑桥,思想发生根本转变;《哲学研究》手稿逐步成形 |
| 二战期间 | 曾在伦敦盖伊医院(Guy's Hospital)从事勤杂/护理工作 |
| 1939 | 接替摩尔任剑桥哲学教授 |
| 1946–1949 | 撰写构成《哲学研究》第二部分的内容,形成晚期思想独特阶段 |
| 1947 | 辞去剑桥教职,专心写作 |
| 1951年4月29日 | 因前列腺癌在剑桥去世,遗言"告诉他们,我度过了美好的一生"(Tell them I've had a wonderful life) |

## 近年维特根斯坦研究学术动向(替代"最近12个月动态")

- **卑尔根维特根斯坦档案馆(WAB)数字化进展**:WAB 自2014年起持续提供 Wittgenstein Nachlass 的数字影印本(通过 Wittgenstein Source 网站免费开放),2016年起在 IDP(交互动态展示)与 SFB(语义分面检索)平台开放全部转录文本的交互式访问。2022–2024年由 Alois Pichler 与 Michele Lavazza 主持的图形重绘项目,重制约1000份图像文件并嵌入转录文本;近期工作聚焦转录校订、逻辑—数学符号呈现改进及缺失手稿影印的补全,WAB 并计划与牛津大学出版社推出修订版《卑尔根电子版》(Bergen Electronic Edition)。
- **"维特根斯坦文库2.0"研讨会**:2024年12月12–13日于维也纳维特根斯坦故居举行,聚焦 Nachlass 相关的最新技术手段、学术交流与公众推广方向。
- **新版笔记本校勘出版**:学者 Marjorie Perloff 编译的《Private Notebooks: 1914–1916》(英译对照本)首次将其一战私人笔记本完整译介英语学界,2025年*Nordic Wittgenstein Review* 刊出相关书评,凸显该笔记本与《逻辑哲学论》成书过程关系的最新研究热点。
