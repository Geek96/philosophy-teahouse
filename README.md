# 哲学茶话会

哲学茶话会是一套可在 Codex session 中长期调用的多人思想对话协议。

当你说“我们来开茶话会”时，店小二会入场主持，依次确认茶会类型、茶温和茶题。随后，苏格拉底、尼采、加缪、孔子、庄子、维特根斯坦会根据各自的思想张力选择发言或沉默。

## 快速开始

```text
我们来开茶话会
```

店小二会问：

```text
店小二：客官，诸位先生已入座，茶炉也温着。今日这席茶，是想解一桩心事，辩一个题目，读一页书，还是看一件人间事？
```

也可以直接带主题：

```text
我们来开茶话会，我最近在职业选择上很犹豫。
```

```text
我们来开茶话会，辩题是：人应该优先追求真实自我，还是承担关系中的责任？
```

## 四种模式

- 共渡难关：陪你处理烦恼、焦虑、人生卡点和选择困难。
- 圆桌讨论：讨论热门话题、哲学问题、社会现象和技术趋势。
- 读书观察：聊一本书、一段文字、生活观察、影视作品或人物现象。
- 二元辩论：围绕明确辩题进行正反方辩论。

## 茶温

- 温茶：陪伴、承接、澄清优先。
- 热茶：默认茶温，有思辨、有反驳、有建设性。
- 烈茶：强质询、强辩论、强拆解。

## 目录

- `AGENTS.md`：茶话会总协议。
- `personas/`：六位人物的独立档案。
- `templates/`：四种茶会模式的可复用模板。
- `sessions/`：未来的茶话会记录。

## 核心原则

共享话题，不共享人格。

人物可以看到同一场讨论，但每个人只能使用自己的思想模型、表达方式和边界。没有独特贡献时，人物可以沉默。
